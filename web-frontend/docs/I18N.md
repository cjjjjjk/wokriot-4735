# hệ thống đa ngôn ngữ (i18n)

## tổng quan
hệ thống hỗ trợ chuyển đổi giữa tiếng Anh (EN) và tiếng Việt (VI) với:
- lưu lựa chọn ngôn ngữ vào localStorage
- không bị mất khi reload/refresh page
- nút toggle đẹp mắt với skeuomorphism design
- dễ dàng thêm ngôn ngữ mới

## cấu trúc

```
src/
├── contexts/
│   └── LanguageContext.tsx    # context quản lý ngôn ngữ & translations
├── components/
│   ├── LanguageToggle.tsx     # nút chuyển đổi ngôn ngữ
│   └── LanguageToggle.css     # styling cho nút toggle
```

## cách sử dụng

### 1. import và sử dụng hook
```tsx
import { useLanguage } from '../contexts/LanguageContext';

const MyComponent = () => {
    const { language, setLanguage, t } = useLanguage();
    
    return (
        <div>
            <h1>{t('myKey')}</h1>
            <button onClick={() => setLanguage('en')}>english</button>
            <button onClick={() => setLanguage('vi')}>tiếng việt</button>
        </div>
    );
};
```

### 2. thêm nút toggle UI
```tsx
import LanguageToggle from '../components/LanguageToggle';

<LanguageToggle />
```

### 3. thêm translations mới

mở file `src/contexts/LanguageContext.tsx` và thêm key vào cả 2 ngôn ngữ:

```tsx
const translations = {
    en: {
        'myKey': 'my text in english',
        // ...
    },
    vi: {
        'myKey': 'văn bản tiếng việt',
        // ...
    }
};
```

## các key hiện có

### login page
- `login.welcome` - tiêu đề chào mừng
- `login.subtitle` - phụ đề
- `login.email` - label email
- `login.password` - label mật khẩu
- `login.submit` - nút đăng nhập
- `login.success` - thông báo thành công
- `login.error` - thông báo lỗi

### dashboard
- `dashboard.title` - tiêu đề trang
- `dashboard.welcome` - lời chào
- `dashboard.logout` - nút đăng xuất

### toast messages
- `toast.serverError` - lỗi server
- `toast.networkError` - lỗi mạng

### common
- `common.loading` - đang tải
- `common.close` - đóng
- `common.save` - lưu
- `common.cancel` - hủy
- `common.confirm` - xác nhận
- `common.delete` - xóa
- `common.edit` - sửa
- `common.search` - tìm kiếm

## thêm ngôn ngữ mới

để thêm ngôn ngữ mới (ví dụ: tiếng Pháp):

1. cập nhật type `Language`:
```tsx
type Language = 'en' | 'vi' | 'fr';
```

2. thêm translations:
```tsx
const translations = {
    en: { ... },
    vi: { ... },
    fr: {
        'login.welcome': 'Bienvenue',
        // ...
    }
};
```

3. cập nhật `LanguageToggle.tsx` để thêm nút FR

## lưu ý
- translations được lưu trong context, không cần file riêng
- ngôn ngữ mặc định: tiếng Việt (vi)
- lựa chọn được lưu trong localStorage với key: `language`
